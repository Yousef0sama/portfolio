# Yousef osama portfolio project
A modern and fast portfolio built with Next.js, TailwindCSS, and optimized for performance and SEO

## Features
- Fast performance with SSR
- Fully responsive UI
- Smooth animations
- dynamic project pages
- fully dynamic wesite
- all data is fetched from a real API connected to a data base

## Tech Stack
 - Next.js 16
 - TailwindCss
 - TypeScript
 - SubaBase database
 - Netlify Host
